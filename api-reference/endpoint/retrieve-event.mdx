---
title: "Retrieve event"
description: "Retrieve a single ingested event."
openapi: "GET /v1/events/{event_id}"
---

## Request

```http
GET /v1/events/12345
```

## Response

```json
{
  "id": "12345",
  "object": "event",
  "user_id": "user-abc123",
  "item_id": "7f3a2c9e",
  "type": "click",
  "click": {},
  "session_id": "4a93",
  "request_id": "c7f0d2bb-56b1-4ca7-8f09-84d8e4ed02b4",
  "context_id": "101",
  "placement": "home_feed",
  "metadata": {
    "context_id": "101",
    "click": {}
  },
  "occurred_at": 1777478400,
  "created": 1777478405
}
```

`session_id`, `request_id`, and `placement` are `null` when not provided at ingest time.

## Errors

| Status | Scenario |
| --- | --- |
| `400` | Malformed event ID |
| `404` | Event does not exist for the authenticated tenant |
