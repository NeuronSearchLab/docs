---
title: "Delete item"
description: "Delete a catalog item."
openapi: "DELETE /v1/items/{item_id}"
---

## Request

```http
DELETE /v1/items/7f3a2c9e
```

## Response

```json
{
  "message": "Item deleted successfully",
  "itemId": "7f3a2c9e",
  "deletedCount": 1,
  "processing_time_ms": 42
}
```

## Errors

| Status | Scenario |
| --- | --- |
| `400` | Malformed item ID |
| `401` | Missing or invalid Bearer token |
| `403` | Token does not include `neuronsearchlab-api/write` |
