## Description

Creates one or more catalog items. Use `POST /v1/items/{id}` to update an existing item.

## Request

```http
POST /v1/items
```

```json
{
  "id": "itm_7f3a2c9e",
  "name": "Wireless Headphones",
  "description": "Noise-cancelling Bluetooth headphones.",
  "metadata": {
    "category": "electronics",
    "brand": "Acme",
    "price": 10999,
    "currency": "usd"
  }
}
```

`id` is optional. If omitted, NeuronSearchLab generates an `itm_` prefixed ID.

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `id` | string | no | Prefixed item ID such as `itm_7f3a2c9e`. |
| `name` | string | yes | Human-readable item name. |
| `description` | string | no | Text used for embeddings and debugging. |
| `metadata` | object | no | JSON object with filterable catalog attributes. |

## Response

```json
{
  "id": "itm_7f3a2c9e",
  "object": "item",
  "name": "Wireless Headphones",
  "description": "Noise-cancelling Bluetooth headphones.",
  "metadata": {
    "category": "electronics",
    "brand": "Acme",
    "price": 10999,
    "currency": "usd"
  },
  "active": true,
  "created": 1777478400,
  "updated_at": 1777478400
}
```

## Errors

| Status | Scenario |
| --- | --- |
| `400` | Missing body, invalid JSON, missing `name`, or malformed `id` |
| `401` | Missing or invalid Bearer token |
| `403` | Token does not include `neuronsearchlab-api/write` |
| `409` | An item with the requested ID already exists |
