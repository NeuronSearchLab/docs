Routes that manage the tenant's account, credentials, billing relationship, and ingest plumbing.

## API keys

See the [API Keys guide](/guides/api-keys) for the full lifecycle.

| Method | Path | Purpose |
| --- | --- | --- |
| `GET` | `/api/api-keys` | List platform API keys |
| `POST` | `/api/api-keys` | Create platform API key |
| `PATCH` | `/api/api-keys/{id}` | Update platform API key metadata/scopes |
| `DELETE` | `/api/api-keys/{id}` | Revoke platform API key |
