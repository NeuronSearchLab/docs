Routes that manage the tenant's account, credentials, billing relationship, and ingest plumbing.

## API keys

See the [API Keys guide](/guides/api-keys) for the full lifecycle.

### List API keys

```http
GET /api/api-keys
```

<CodeGroup>

```bash cURL
curl "https://console.neuronsearchlab.com/api/api-keys" \
  -H "Authorization: Bearer nsl_<prefix>_<token>"
```

</CodeGroup>

### Create API key

```http
POST /api/api-keys
```

<CodeGroup>

```bash cURL
curl -X POST https://console.neuronsearchlab.com/api/api-keys \
  -H "Authorization: Bearer nsl_<prefix>_<token>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Staging Backend",
    "scopes": ["recommendations", "events", "items"],
    "environment": "staging"
  }'
```

</CodeGroup>

### Update API key

```http
PATCH /api/api-keys/{id}
```

<CodeGroup>

```bash cURL
curl -X PATCH https://console.neuronsearchlab.com/api/api-keys/key_42 \
  -H "Authorization: Bearer nsl_<prefix>_<token>" \
  -H "Content-Type: application/json" \
  -d '{ "name": "Production Backend", "environment": "prod" }'
```

</CodeGroup>

### Revoke API key

```http
DELETE /api/api-keys/{id}
```

<CodeGroup>

```bash cURL
curl -X DELETE https://console.neuronsearchlab.com/api/api-keys/key_42 \
  -H "Authorization: Bearer nsl_<prefix>_<token>"
```

</CodeGroup>
