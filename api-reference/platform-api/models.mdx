Routes for the full training-to-serving lifecycle. See the [Models guide](/guides/models) for the broader workflow.

## Model versions

### List models

```http
GET /api/models
```

<CodeGroup>

```bash cURL
curl "https://console.neuronsearchlab.com/api/models" \
  -H "Authorization: Bearer nsl_<prefix>_<token>"
```

</CodeGroup>

### Create model metadata

```http
POST /api/models
```

<CodeGroup>

```bash cURL
curl -X POST https://console.neuronsearchlab.com/api/models \
  -H "Authorization: Bearer nsl_<prefix>_<token>" \
  -H "Content-Type: application/json" \
  -d '{ "name": "homefeed-v3", "family_id": 2, "description": "Q2 retraining" }'
```

</CodeGroup>

### Retrieve model

```http
GET /api/models/{id}
```

<CodeGroup>

```bash cURL
curl "https://console.neuronsearchlab.com/api/models/55" \
  -H "Authorization: Bearer nsl_<prefix>_<token>"
```

</CodeGroup>

### Update model


```bash cURL

  -H "Authorization: Bearer nsl_<prefix>_<token>" \
  -H "Accept: text/event-stream"
```
